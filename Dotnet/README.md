# ASP.NET Core MVC — MvcMovie

[Tutorial](https://learn.microsoft.com/en-us/aspnet/core/tutorials/first-mvc-app/start-mvc): model Movie (Title, ReleaseDate, Genre, Price), MoviesController, widoki CRUD, EF Core + SQLite. Przy pierwszym runie tworzy się MvcMovie.db.

.NET 9 SDK.

```bash
cd Dotnet
dotnet run
```

Adres w konsoli — otwórz w przeglądarce, w menu **Movies**. Lista (pusta na starcie), Create New, Edit, Details, Delete.
